Realms of Ash CX

Realms of Ash CX is a compact fantasy roguelike RPG built in C for the TI-Nspire CX / CX II calculator using Ndless.

Explore a ruined ash-covered world, fight enemies, unlock new regions, collect keys, upgrade your gear, craft supplies, clear sub-dungeons, and survive long enough to challenge the Lord of Ash. The game is designed to run within calculator limits while still offering a surprising amount of RPG content.

Gameplay Overview

The player begins in the Ash Wastes and must explore, hunt, collect supplies, and find keys to unlock new regions. Each new area introduces stronger enemies, better rewards, and new dangers.

Progression is built around exploration, combat, resource management, and preparation. Players can upgrade weapons, armor, and charms, craft useful items, complete repeatable Quest Board contracts, and search for rare events hidden throughout the ashland.

After defeating the final boss, the game unlocks Ash Trials, giving players a post-game challenge loop with stronger enemies, Trial Champions, and the Echo of the Lord.
